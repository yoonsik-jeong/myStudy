01-1.시스템개요

  • 시스템개요

    공통의 목적을 위해 여러가지 상호 요소들이 유기적으로 결합

  • 시스템 특성

    • 목적성 : 시스템을 구성하는 각 요소들이 어떤 하나의 최종 목표를 향해 도달하고자 하는 특성

    • 자동성 : 어떤 조건이나 상황의 변화에 대응하여 스스로 대처하는 특성

    • 제어성 : 지정된 규정이나 한계, 또는 궤도로부터 이탈하는 사태나 현상의 발생을 사전에 감지하여 올바르게 수정하는 특성

    • 종합성 : 다른 시스템들이 상호 의존 관계로 통합되는 특성

  • 시스템 기능

    • 입력 기능 : 처리할 데이터, 처리 조건, 처리 방법을 시스템에 투입

    • 처리 기능 : 처리할 데이터를 처리 조건이나 방법에 따라 변환(가공)하는 것

    • 출력 기능 : 출력된 데이터를 시스템에 산출하는 것

    • 피드백 : 출력된 데이터가 목적을 만족하지 못한 경우, 목적을 만족할때까지 반복 처리하는 과정

    • 제어 기능 : 입력부터 출력까지 과정을 감시하며, 각 과정에서 생성되는 조건을 판단하여 돌발사태를 처리하는 안정화 기능까지 포함
