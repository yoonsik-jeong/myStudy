08.코드의 문서화

  • 정의

    코드를 실제 사용할 프로그래머들이 이해할 수 있도록 코드와 관련한 전반적인 내용을 문서로 작성하는 과정

    • 코드 설계서(코드 설정 기준서)

      코드 명칭, 자릿수 및 코드 체계등을 기술

    • 코드표

      코드 설계서(코드 설정 기준서)에 의하여 구체적으로 코드를 할당한 내용을 기록하고, 코드표는 관련 부서의 실무 담당자가 사용 할 수 있도록 한다.

    • 코드 명세서(코드 대장,코드 파일)

      최종적으로 코드를 총괄하는 부서에서 보관하는 문서로, 코드 설계서 및 코드표와 함께 코드의 사용과 유지보수에 필요한 세부 지침 등을 자세히 기록한 관리 기준이 포함
